Comparison techniques used in this study:

In order to visualize the previously explained changes, we will provide three types of comparison techniques named as "Side by side", "Animation" and "Reduced side by side". In this tutorial each of these comparison methods will be brifely explained. It is important to note that you will be shown only one change and therefore you will be asked to detect only one type of change in each question.

Animation:

One way of showing the changes is representing the changes using animated transitions. In order to observe these changes we provided "start" and "reset" buttons. When you click on "start" button you will observe animated transition of a change in the given visualization and using "reset" button you can return to the original state of the visualization. Reduced side by side

In this comparison method you will see three parts of visualization in one screen side by side. The left panel which is named as "current visualization" shows the current state of visualization after the change is applied to visualization. The two other parts in the left hand side of "current visualization" shows only part of the visualizaiton which is affected by the change and filters out other nodes. The part where it is named as "before" is part of visualization which will undergo change and the part named "after" shows the changed entity and its immediate parent. See example below:

As can be seen node "P" is added to node "G" in this example and in the right panels we only show the immediate parent of "P" which is "G".
